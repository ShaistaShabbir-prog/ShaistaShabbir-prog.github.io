Undergraduate Final Year Project · UMSIT Kotli AJK

ALSK — Arms Licensing System Kotli

BSCS Final Year Project UMSIT Kotli AJK 2015–2016

Web-based arms licensing management system developed for the Licensing and Security Department of District Kotli, AJK. Digitised the manual licensing process with CNIC-based access, QR-code embedded license cards, online verification for check posts, and expiry date alerts.

System details & features

Problem it solved: The existing manual system had critical issues — large file volumes, poor security, no backup, and very slow search. ALSK replaced this with a fully digital, multi-tier web application.

Key features:

  • CNIC-based information access and manipulation
  • Online license verification for check post investigators
  • QR code embedded license cards (card format instead of booklet)
  • Search by CNIC, District, City, or Village
  • Automatic expiry date alerts
  • Multi-role access: Admin, Applicant, and Post Investigators
  • Online license application forms

System design: OMT (Object Modelling Technique), relational database, class diagrams, use case diagrams, sequence diagrams, and activity diagrams.

Stakeholders: Government of District Kotli (DC Office), State AJK, License Applicants, Check Post Investigators.
