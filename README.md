## 🔬 Current Research

### AudioIno (Flagship)
Expert-guided annotation platform for machining stability and chatter analysis. Combines domain expert knowledge with structured data workflows for high-quality labeled machining datasets.

### Low-Cost Acoustic Monitoring for CNC Milling
Physics-informed evaluation of low-cost microphone systems (PCB, B5, HI10, TASCAM) — spectrogram similarity, dominant frequency tracking, impact hammer testing.

### Tool Wear Prediction
ML workflows combining CNN, Random Forest, feature engineering, and explainability for tool-condition monitoring.

### Structural Dynamics & FRF Analysis
Frequency Response Function analysis using H1/H2 estimators for resonance identification and chatter mechanism characterization.
