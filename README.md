## Current Research — AudioIno

AudioIno is an expert-guided platform and analysis workflow for:
- **Machining stability annotation** (stable / metastable / chatter labels)
- **Acoustic process monitoring** using PCB, B5, HI10, TASCAM microphones
- **Spectrogram-based analysis** — STFT, Welch PSD, dominant frequency tracking
- **FRF / Impact hammer analysis** for resonance-guided signal interpretation
- **Data-driven manufacturing intelligence** — reproducible ML-ready datasets

## Design Principles

- **Academic first** — profile for PhD supervisors, professors, conference reviewers
- **Honest** — no invented publications; work-in-progress clearly labelled
- **Clean hierarchy** — current TU Dortmund research dominates; prior work secondary
- **Dark mode** — full CSS `prefers-color-scheme: dark` support
- **Accessible** — semantic HTML, proper heading hierarchy, keyboard navigable
- **Fast** — no JS frameworks, no build tools, loads in < 1 second
